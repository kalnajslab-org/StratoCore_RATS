
## RATS Telemetry Messages

| Purpose | Msg1 | Flag1 | Msg2 | Flag2 | Msg3 | Flag3 | C++ Structures | Payload Contents |
|---------|------|-------|------|-------|------|-------|----------------|------------------|
| RATS data | **RATSReport** | FINE | \<mode\> | FINE | \<lat,lon,alt\> | FINE | `RATSReport_t`, `ECUReport_t` | Metadata header, then ECU data blocks |
| General text | **RATSTextTM** | INFO | \<mode\> | FINE |Text message| FINE | `RATSTextTM_t` | Text message |
| RATSEEPROM | **RATS EEPROM data; length \<n\>** | FINE | | | | | `RATSEEPROM_t` | Raw RATS EEPROM data; length in flag field |
|**MCB Related TMs** | | | | | | | | |
|_Need to add the TMs generated by the MCB_ | | | | | | | |See SendMCBTM() |
| MCB Motion | **MCB TM (Packet \<n\>)** | FINE | Reel: \<pos\> | | | | | Motion data, optional sync/time header, reel position |
| MCB EEPROM | **MCB EEPROM Contents** | FINE | | | | | `MCBEEPROM_t` | Binary MCB EEPROM dump |
